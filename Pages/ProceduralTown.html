Procedural Town


General Information

Summary: Procedural Town is the project name for a procedural town generator inspired by Morthal. Using a simple in-game created drawing tool you can draw paths & water and use all kinds of sliders to create your perfect city!
Coding Platform: Unity
Target Platform: PC
Development Time: 10 Weeks (2022)
Solo Project

Description & Contributions

This simple tool allows for the generation of entire cities! It was created to learn more about procedural mesh creation, placement and tool creation in Unity.

My Contributions
All programming aspects were done by me. All art textures and foliage are free assets from the asset store. House model creation is procedural and done in code. The key points of this project are:
- Drawing & image processing in Unity
- Procedural terrain warping & creation
- Water & Snow shader creation in HDRP
- Procedural optimized mesh generation (+UVs)
- House creation algorithm
- House placement algorithm based on path placement
