UELab


General Information

Summary: UELab is a small test environment with 2 levels to test out the different features of Unreal. This was my first introduction to the engine.
Coding Platform: Unreal Engine 5 - Blueprints
Target Platform: PC
Development Time: 2 weeks (2022)
Solo Development

Description & Contributions

This was a personal project created to get some experience in the Unreal Engine. It was my first unreal game, and many basic features like use of actors, pawns, and blueprints were explored.

My Contributions
My focus during the project was on general gameplay, physics and just exploring as many things as possible. My main learning points of the project were:
- Player controller creation
- Bow & held object physics
- Platforming physics
- UI (widgets), Sound integration, art integration & simple scene building
