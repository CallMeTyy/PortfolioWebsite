UELabVR


General Information

Summary: UELabVR is a small arena-fighter game which served as a learning project for Unreal VR. In the arena you fight ragdolling enemies using swords, a bow or your bare fists!
Coding Platform: Unreal Engine 5 - Blueprints
Target Platform: PCVR
Development Time: 4 weeks (2022)
Solo Development

Description & Contributions

This was a personal project created to get some experience in the Unreal Engine. It was my first VR game, and many features like the physics engine, animation, ragdolls and other aspects were explored. It was created using blueprints and the VRExpansionPlugin.

My Contributions
My focus during the project was on general gameplay, physics and just exploring as many things as possible. Together with free art assets and the VRExpansionPlugin for Unreal I created many different interactions and learned a lot about unreal. My main learning points of the project were:
- VR player controller integration
- Sword & Bow physics
- Character animations & ragdolls
- UI (widgets), Sound integration, art integration & simple scene building
