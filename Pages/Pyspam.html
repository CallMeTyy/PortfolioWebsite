Python Spam Filter


General Information

Summary: A simple e-mail classifier made in Python to learn the language. It can train with a given input of classes and return a data document, which can be used to evaluate and classify emails.
Coding Platform: Python, Docker
Target Platform: PC/Server
Development Time: 8 weeks
Solo Project

Description & Contributions

This project was my introduction to Python. I made all aspects, with minimal use of libraries.

My Contributions
I was the lead engineer of the project. Together with a company coach, I wrapped two python programs into a docker and created a simple command-line run program that runs both and outputs the result. Everything was wrapped into a docker compose for easy access and use on a windows machine (destined for linux). The key points of this project are:
- Utilizing python to use several deepfake creation libraries and programs
- Creating a simple command line interface and the possibility for web call interactions
- Utilizing docker to create a simple to download and install program
- Working in a small team to create a simple application
- Utilizing several AIs to create audio and syncing videos

All projects at the Saxion XR Lab were not allowed to be shared, with exceptions for video showcases. The video below demonstrates a simple version of the program. The web call gets processed, and the voice generator starts working. The video generator waits until the voice is done, and then lip syncs a video or photo to the audio. It outputs it in a folder to be sent to the user.
