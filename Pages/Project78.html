Project 78


General Information

Summary: Project 78 is a puzzle adventure game in Virtual Reality. Play alone or in co-op while solving physics based puzzles with your bow and uncover the mysteries of a new planet.
Coding Platform: Unreal
Target Platform: PCVR
Development Time: 20 weeks (2023-2024)
Team (8 people):
- Artists: Vy Nguyen, Kevin Nijkamp, Jeroen Vedder and Quint Winter
- Technical Artist: Teun de Vries
- Designer: Ira Evers
- Engineers: Maurits Dijkman and Tycho Tuitert

Description & Contributions

This was a large project for a project called IMT&S we undertook initially as a continuation of Apollo Arena. We quickly discovered that we wanted to do something different this time, and went for a whole different type of game. We created a demo for our game in which the gameplay is clear, and it forms the first level for our game. We might continue this project in the future, so feel free to visit this page from time to time to see if there is progress.

My Contributions
My focus during the project was on general gameplay, physics and networking. I worked on the following things:
- VR Physics (General + Arrow combinations)
- Bow & Arrow Physics
- Different arrow types and interactions
- Puzzle implementations
- Enemy interactions
- Player IK (inverse kinematics) rig (hands & legs)
- Syncing everything over the network
