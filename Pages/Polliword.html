Polliword


General Information

Summary:Polliword is a simple puzzle platformer inspired by baba is you in a custom physics engine. You, the frog, have to get to the end of all three levels by eating "Word Flies" that can change the physics of the level!
Coding Platform: GXP Engine (C# .NET)
Target Platform: PC
Development Time: 2 Weeks (2021)
Team (7 people):
- Artists: Bram Verschuren, Dilyana Parvanova and Nikoleta Staikova
- Designers: Ira Evers, Jan de Weijer and Teun de Vries
- Engineer: Tycho Tuitert

Description & Contributions

Polliword is a game made in GXP engine, a barebones in-house engine from my study, combined with personally made physics. It was made as a group effort for a school project in which we were tasked to create a heavily physics-focused game.

My Contributions
All programming aspects were done by me. I created all physics, and created a tool to integrate levels using tiled. All art and audio implementation was also done by me. The key points of this project are:
- Creating physics from scratch for a 2D game
- Creating a game using only C# .NET
- Visual integrations (level integration, art asset integration, animations, tilemaps etc.)
- Audio integration
- Player controller
- Word fly physics (changing the world using flies)
- Grappling hook physics
- Swimming physics
