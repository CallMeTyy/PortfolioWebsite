Vector Arcade


General Information

Summary: Vector Arcade is a simple showcase of several physics interactions I made from scratch in GXP Engine (C# .NET).
Coding Platform: GXP Engine (C# .NET)
Target Platform: PC
Development Time: 8 weeks
Solo Development

Description & Contributions

This game showcases several physic interactions between lines, cubes and circles. These were combined to make a primitive platformer and a bullet hell game using vector graphics.

My Contributions
All aspects were made by me, the important points from this project are:
- Creating a 2D physics engine from scratch
- Creating gamified experiences from the physics engine
- creating a simple physics platformer with explosions and lines
- creating a bullet hell game where enemies spawn infinitely
- creating a vector styled game
