FearVR

Project Code TLOF_7


General Information

Summary: FearVR is a proof of concept made for an artist named Farah Stretah. It combines Stable Diffusion (AI) with an eerie environment in VR to generate fear through user submitted prompts.
Coding Platform: Unity
Target Platform: PCVR
Development Time: 2 Weeks
Created in collaboration with: Ira Evers

Description & Contributions

FearVR was created in a temporary work collaboration with artist Farah Stretah to give a proof of concept for one of her art projects. In VR, the user is teleported into a space environment in which an eerie sound guide transforms your world into a calm environment. The player then moves up slowly towards the screens where they will be surrounded by AI generated images. Bystanders can use their (android) phones to send in prompts which the user will endure. The goal is to provoke fear.

My Contributions
I worked as a programmer on this project, which I joined at a late stage of development. I was responsible for the networking, the VR program and the events that connect everything. My collaborations are:
- Creating a simple VR player
- Testing different scenarios and building the environment
- Creating several events based on audio and time to guide the user through the program
- Explore Stable Diffusion and integrate it in Unity
- Explore ChatGPT and integrate it in Unity
- Create a simple networking connection between phones and the program using OSC
- Creating a simple queue that stores all prompts sent by the bystanders
