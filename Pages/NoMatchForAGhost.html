No Match For A Ghost


General Information

Summary: No Match for a Ghost is a VR detective puzzle where the player is tasked to find out why a ghost is haunting a house. In this light-hearted horror-type game your only light source is a match, which runs out pretty quickly. Can you find out how to free her?
Coding Platform: Unity
Target Platform: PCVR
Development Time: 4 days (Game Jam)
Developed together with: Teun de Vries

Description & Contributions

This was a game jam project with the theme "One-Time Use". Our implementation of the theme was the match that runs out quickly, only providing a little bit of light with each use. This provided an eerie atmoshphere perfect for a detective horror game. While the setting is scary, we focussed on making it fun and light hearted without any jumpscares or too scary elements to make it safe for everyone.
