Coastal Cleanup


General Information

Summary: Coastal Cleanup is a simple serious game to teach children to clean up their surroundings! It was made as an effort to help support causes that want to clean the beaches. The player character, a crab, goes around cleaning the beach to score points.
Coding Platform: Unity
Target Platform: PC
Development Time: 3 Weeks
Team (7 people):
- Artists: Bram Vaarwerk and Dilyana Parvanova
- Designers: Ira Evers, Jan de Weijer and Teun de Vries
- Engineer: Tycho Tuitert

Description & Contributions

Coastal cleanup is a small serious game for children. With a fun theme and happy music we wanted to create an immersive and engaging way for children to become aware of the problems in our world, in this case the plastic issues.

My Contributions
During this project I worked as the programmer, focussing mainly on making everything work and creating the procedural animations. My main learning points of the project were:
- Gameplay & Character controller
- Procedural Crab animations using IK (legs & hands)
- Pickup physics
- Score detection and UI integration (pause/main menu)
