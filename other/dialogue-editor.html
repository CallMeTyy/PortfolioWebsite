How it works

Each node has one input at the top. Dialogue nodes can have any number of output categories, and every category gets its own exit on the right side.

Use Text for a fixed response, or AI answer when the runtime should generate the reply dynamically.

Controls

Scroll wheel: zoom in and out of the graph.

Middle mouse / wheel press: pan the canvas.

Right click: open the add-node menu at the cursor.

Connecting nodes

Click a category exit on the right of a dialogue node, then click the input port on another node. Click the unlink button beside a category to clear its connection.

Exported JSON

The exported file contains node positions, titles, response mode, response content, output categories, and their target node IDs, so you can reload or process it elsewhere.
