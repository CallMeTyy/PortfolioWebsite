Building Blocks - Rating my building block

Having now finished most aspects of the building block, a modular power block that can be used to create a variety of configurations in regards to power strips and power, the block can be reviewed based on what I learned in the Mastering Tinkering course.

In the Tinker Materials and Tinker Materials 2 sections, I discuss the materials and techniques used in the tinkering process. These form the base of what I will rate my building block on.

Properties

In the tinkering materials sections, I discuss the properties of the materials used in the building block. These properties include:

  • Affordance: Self Guiding: Whether the building block is designed to guide the user.
  • Affordance: Previousness: Whether the building block has value and can or cannot be broken or modified to fit needs by altering it physically.
  • Affordance: Skill needed: Whether the building block requires a certain level of skill to assemble and use effectively.
  • Affordance: Interoperable: Whether the building block can be used with other tinkering tools/things.
  • Novelty: Whether the building block introduces new ideas or approaches
  • Low threshold, high ceiling: Whether the building block is easy to start with but offers opportunities for advanced use.
  • Wide walls: Whether the building block can be used in a variety of ways and for different purposes.
  • Price: Whether the building block is affordable.
  • Recyclability: Whether the building block can be recycled or repurposed at the end of its life or when finished using it.
  • Durability: Whether the building block is built to last and can withstand regular use.
  • Overall sustainability: Whether the building block is overall environmentally friendly.
  • Strength: Whether the building block is structurally strong and can support the intended use.
  • Storage (Size & Weight): Whether the building block can be stored efficiently and is not too heavy.
  • Cost (Manufacturing): Whether the building block is cost-effective to produce.
  • Alteration & Repair: Whether the building block can be modified or fixed if damaged.
  • Availability: Whether the building block is readily available and easy to obtain.

As a short pre-review, I think the building block in terms of usage for this course is not ideal. It's not really tinkerable asin usable to create new unimagined things, it has very limited walls and while being a very useful product, does not really allow for a tinkering mindset. I now realize that the building block is more of a product than a tinkering tool, and while it can be used in a tinkering process, it does not really fit the criteria of a tinkering tool. With the new views of what should be a tinkerable tool the building block will however still be reviewed.


Affordance: Self Guiding

The building block would probably need a very short guide that is on the box, with additional instructions inside a booklet in order to be self-guiding. At the moment you'd need to know what the connector piece is and how to use it, and once you know that you can easily figure out the rest. Some aspects of its interoperability need some further instructions. It's simple which makes it easy to understand.

Affordance: Previousness

The building blocks are not really something that need to be altered. The cords between the blocks would be nice if easily cuttable to customize the length, but this is not something easily realized. It's thus a very precious product which makes it hard to use for the full imagination.
